    FAQ

    1. Что в этом тестовом задании вам было сделать легко, что тяжело?

      Особых трудностей при выполнении задания не возникло. Есть только нерешенный момент с градиентными рамками кнопки, при наличии времени, смогу это исправить;

    2. Какие вспомогательные инструменты(библиотеки, плагины, npm-пакеты, сервисы и прочее…) вами использовались и для чего?

      Библиотека jquery - для селектора (некоторые события и вспомогательный для плагина скролла);

      Плагин jquery slimscroll (для кастомизациискролла в селекте);

      Пакетный менеджер npm (пакеты в папке node modules: для компиляции SCSS-файлов, для автопрефиксов, для создания иконочного шрифта, для минификации картинок, css-файлов и js-файлов);

      Сборщик gulp для сборки проекта и работы с npm пакетами;

    3. Где вы применили один механизм, но при этом знаете какой-то другой и почему вы выбрали именно этот?

      В селекте я использовала список и jquery, но можно еще выполнить с помощью тега select. Вариант со списками первый пришел в голову;

    4. Какие из задач тестового задания вы бы реализовывали по разному – будь это большой проект на 100+ страниц или одностраничный сайт?

      Для больших сайтов лучше использовать компонентный подход (BEM-методология), чтоб при необходимости легко повторять элементы на разных страницах и с разным позиционирование;

    5. Если где-либо в верстке использовались картинки(спрайт или отдельные картинки), можно ли было бы это же реализовать на чистом CSS?

      Не совсем понятен вопрос;
